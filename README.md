## Mis estudios en Kotlin

### Integrar IA en Android Studio

Para facilitar el uso de Kotlin en Android Studio, primero se recomienda
integrar herramientas de inteligencia artificial:

    Ir a ⚙️ → Settings → Tools → AI → Activar "Use all Gemini" → OK → Apply → OK

------------------------------------------------------------------------

### Instalar plugin desde archivo en Android Studio

A continuación, puedes instalar plugins manualmente:

    Ir a ⚙️ → Settings → Plugins → Install plugin from disk → 
    Seleccionar el archivo de instalación → OK → Apply → OK

------------------------------------------------------------------------

### Plugins esenciales para Kotlin

#### 1. Kotlin (oficial)

-   Generalmente ya viene integrado.
-   Proporciona autocompletado, depuración y soporte completo.
-   Es básico para trabajar con Kotlin.

#### 2. JSON to Kotlin Class

-   Convierte JSON a `data class` automáticamente.
-   Ideal para trabajar con APIs (por ejemplo, Retrofit).

Ejemplo: pegas un JSON y genera la clase automáticamente.

#### 3. Key Promoter X

-   Enseña atajos de teclado mientras programas.
-   Ayuda a mejorar la velocidad y reducir el uso del mouse.
-   Recomendado para principiantes.

#### 4. Codota / TabNine (IA)

-   Autocompletado inteligente basado en inteligencia artificial.
-   Sugiere bloques completos de código.
-   Aumenta significativamente la productividad.

#### 5. ADB Idea

-   Permite ejecutar comandos ADB sin usar la terminal.
-   Ejemplos: reiniciar la app, limpiar datos, etc.
-   Muy útil para pruebas.

------------------------------------------------------------------------

### Plugins para escribir mejor código

#### 6. String Manipulation

-   Permite transformar texto rápidamente:
    -   camelCase
    -   snake_case
    -   entre otros formatos
-   Ahorra tiempo en refactorización.

#### 7. Rainbow Brackets

-   Colorea paréntesis y llaves: `{}`, `()`.
-   Facilita la lectura de código complejo.

------------------------------------------------------------------------

### Plugins visuales (mejor experiencia)

#### 8. Material Theme UI

-   Cambia el tema del IDE.
-   Mejora la experiencia visual y reduce la fatiga.

#### 9. CodeGlance

-   Muestra un minimapa del código (similar a VS Code).
-   Permite navegar rápidamente en archivos grandes.

------------------------------------------------------------------------

### Bonus: plugins útiles

-   **GitToolBox**: mejora la integración de Git en Android Studio.
-   **Android Drawable Preview**: permite visualizar imágenes sin
    abrirlas.
-   **Android Drawable Importer**: facilita la importación de íconos.

Todos estos plugins ayudan a optimizar el flujo de trabajo y ahorrar
tiempo.
